# Images Directory

Place your carousel images here:

1. `placeholder-1.jpg` - Your first carousel image
2. `placeholder-2.jpg` - Your second carousel image
3. `placeholder-3.jpg` - Your third carousel image
4. `placeholder-4.jpg` - Your fourth carousel image

## Image Recommendations

- **Format**: JPG or PNG
- **Dimensions**: Square aspect ratio (e.g., 800x800px or 1000x1000px)
- **File size**: Keep under 500KB for optimal loading
- **Content**: Personal photos, activities, or professional shots

## Usage

The carousel will auto-play through these images every 3 seconds. Users can also manually navigate using the left/right arrows or by swiping on mobile devices.

To update the carousel images, edit the `carouselImages` array in:
`components/sections/intro-section.tsx`
